# wexample-helpers-yaml

Helpers for reading yaml files.

Version: 0.0.65

## Requirements

- Python >=3.10

## Dependencies

- pydantic>=2,<3
- pyyaml
- wexample-helpers==0.0.65

## Installation

```bash
pip install wexample-helpers-yaml
```
